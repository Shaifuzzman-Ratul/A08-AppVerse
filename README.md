## Project Overview
AppVerse is a modern React-based web application that allows users to explore, search, and install various applications. The platform provides smooth navigation, live search functionality, and localStorage-powered installation management, giving users a seamless experience to browse and manage apps efficiently.


## Features
- **Browse Featured Apps:** View multiple apps in an organized and visually appealing layout  
- **Live Search:** Search apps in real-time by name  
- **App Details:** View detailed statistics including downloads, ratings, and reviews  
- **Install & Manage Apps:** Install apps and manage them locally via localStorage  
- **Charts & Analytics:** Visualize review data and ratings using interactive charts  
- **Responsive Design:** Fully responsive for desktop, tablet, and mobile  
- **Smooth Notifications:** Success and error feedback using React Hot Toast
